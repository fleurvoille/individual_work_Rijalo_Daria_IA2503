# 6. Реализация требований

## Работа с DOM

Создание элементов:

```javascript
const li = document.createElement("li");
taskList.appendChild(li);
```

Изменение элементов:

```javascript
span.classList.add("done");
```

Удаление:

```javascript
tasks = tasks.filter(t => t.id !== task.id);
```

## Обработка событий

```javascript
addBtn.onclick = () => {
```

## Валидация

```javascript
if (text === "") {
    error.textContent = "Введите задачу";
```

```javascript
let tasks = [];
```
